Tugas Latihan CSS

1. Apa gunanya CSS?

CSS (Cascading Style Sheets) berguna untuk mengatur tampilan dan presentasi sebuah halaman web. Jika HTML berperan sebagai kerangka struktur (tulang), maka CSS berperan sebagai kulit dan baju yang mempercantik.

Kegunaan utamanya meliputi:

  • Mengatur warna, jenis font, ukuran teks, dan latar belakang.
  • Mengatur tata letak (layout), jarak antar elemen (margin/padding), dan posisi elemen.
  • Membuat website menjadi responsif (menyesuaikan tampilan di HP, Tablet, dan Desktop).
  • Memisahkan konten (HTML) dari desain visual, sehingga kode lebih rapi dan mudah dikelola.

2. Implementasi CSS Embed

Pada file ini, saya telah membuat CSS menggunakan metode Embed (Internal CSS). Hal ini bisa dilihat pada bagian <head> di atas, di mana kode CSS ditulis di dalam tag <style> ... </style>, bukan di file terpisah maupun di dalam baris elemen (inline).

3. Setting Tampilan Website

Tampilan website ini telah di-setting sedemikian rupa menggunakan properti CSS:

  • Background: Menggunakan warna biru muda (#e3f2fd) agar nyaman di mata.
  • Container: Menggunakan margin auto agar konten berada di tengah layar dan diberi efek bayangan (box-shadow).
  • Typography: Menggunakan font Segoe UI agar terlihat modern dan bersih.
  • Styling Box: Jawaban ini dibungkus dalam kotak abu-abu dengan garis tepi biru (border-left) untuk memberikan penekanan visual.
